## Getting started
Run the app
```bash
git clone git@github.com:gpspake/memphisww-eslint.git
cd memphisww-eslint
yarn install
yarn start
```
Lint the app
```bash
yarn lint
```
Fix linting errors
```bash
yarn lintfix
```
